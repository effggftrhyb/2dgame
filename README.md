# 🎮 Adventure Quest — 2D Платформер

> **Beta-версия** | Проект находится в стадии активной разработки и тестирования.

## 📖 О проекте

**Adventure Quest** — это 2D платформер, созданный на чистом HTML5 Canvas и vanilla JavaScript без использования сторонних игровых движков. Проект разработан как MVP (Minimum Viable Product) и демонстрирует навыки веб-разработки, игровой логики и работы с графикой.



## 🎯 Цель проекта

- Демонстрация навыков создания интерактивных веб-приложений
- Реализация игровой логики на чистом JavaScript
- Работа с HTML5 Canvas API для рендеринга графики
- Создание полноценного игрового цикла с несколькими уровнями

## 🕹️ Управление

| Клавиша | Действие |
|---------|----------|
| `← →` или `A D` | Движение влево / вправо |
| `ПРОБЕЛ` или `W` | Прыжок |
| `E` | Взаимодействие с NPC |

## 🌍 Уровни

| # | Название | NPC | Артефакт |

## ⚙️ Механики

- **Платформинг** — прыжки по платформам, обход пропастей
- **Сбор монет** — золотые монеты разбросаны по уровням
- **Артефакты** — каждый уровень содержит уникальный артефакт для NPC
- **NPC-квесты** — найди артефакт и отнеси его NPC, чтобы открыть проход на следующий уровень
- **Враги** — крабы, растения и призраки патрулируют уровни; прыгай на них сверху, чтобы уничтожить
- **Жизни** — 3 жизни, при потере всех — Game Over

## 🎨 Используемые ресурсы (атрибутация)

### Спрайты персонажей и врагов
- **Player sprites** — `assets/Player/` — спрайты игрока (idle, walk, jump, attack, hurt)
- **Enemy sprites** — `assets/Enemies/` — спрайты врагов (crab, ghost, plant и др.)
- **Interactive items** — `assets/Interactive/` — монеты, звёзды, ключи, ящики и другие интерактивные объекты

### Наборы тайлов
- **Volcano tiles** — `assets/Tiles/Volcano/` — тайлы для вулканического уровня
- **Milkshake Land tiles** — `assets/Tiles/Milkshake_Land/` — тайлы для кондитерского мира
- **Elementals Ground Monk** — `assets/Elementals_ground_monk_FREE_v1.3/` — бесплатный набор анимаций персонажа (спрайт-лист и GIF-анимации)

### Фоновые изображения
- `1level.png`, `2level.jpeg`, `3level.png`, `4level.png` — фоны для каждого из уровней

### Шрифты
- [Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P) — пиксельный шрифт от Google Fonts

## 🚀 Запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/твой-username/playgame.git
cd playgame
```

2. Откройте `index.html` в браузере (достаточно просто открыть файл, сервер не требуется)

Или запустите локальный сервер:
```bash
npx serve .
```
